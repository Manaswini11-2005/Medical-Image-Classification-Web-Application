📌 Project Overview

This project implements a multi-class brain tumor classification system using MRI images.
The model classifies images into:

- Glioma Tumor  
- Meningioma Tumor  
- Pituitary Tumor  
- No Tumor  

Transfer Learning with MobileNetV2 was used for improved performance.

⚙️ Technologies Used

Python
TensorFlow / Keras
MobileNetV2 (Transfer Learning)
OpenCV
Matplotlib
Scikit-learn

## 📊 Model Performance

- Validation Accuracy: ~66%
- Evaluated using:
  - Confusion Matrix
  - Precision / Recall / F1-score
